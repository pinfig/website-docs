# Layout

Pinfig offers a comprehensive set of tools designed for applying diverse layouts to the note cards within a board.

These layouts serve as automated aids, effortlessly arranging your notes on the canvas in unique patterns.

To use a layout, you can simply brush to select various notes on the canvas and then apply the desired layout.
